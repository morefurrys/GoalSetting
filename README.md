# GOALSETTER WEBAPP

## Usage

Rename the .envexample to .env and add your MONGO_URI

### Install dependencies

```
# Backend deps
npm install

# Frontend deps
cd frontend
npm install
```

### Run Server

```
npm run server
```

### Run App

```
# Locate to GoalSetting folder and run dev
cd GoalSetting
npm run dev

# Locate to your frontend folder and run build
cd frontend
npm run build
```
